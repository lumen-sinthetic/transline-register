# Registration for Tranline international

## To launch project write

```sh
pnpm i

pnpm dev
```

## Implemented steps
- Phone number with country picking
- Role selector
- Phone number verification with otp
- Personal user data form (name, email, e.t.c)
- Minimalistic profile page

## Extra features

- fields validation
- current step and form data seving in cookie files
- localization on `ru` and `en` languages
- toast notifications
- auto focus on error
- dark and light mode

### You can view deployed version [here](https://transline-register.vercel.app)
